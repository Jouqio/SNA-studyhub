#  Getting Started

## Clone Repository

```bash
git clone https://github.com/USERNAME/bimbel-sna-dashboard.git
```

In folder:

```bash
cd bimbel-sna-dashboard
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
